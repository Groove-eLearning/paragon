# Card

<SingleComponentStatus
  frontmatter={
    get(props, 'pageContext.frontmatter') ? props.pageContext.frontmatter : null
  }
/>

##### Basic Usage

```jsx live
<Card style={{ width: '18rem' }}>
  <Card.Img variant="top" src="https://source.unsplash.com/400x200/?nature,flower" />
  <Card.Header>
    <Card.Header.Title>Card title</Card.Header.Title>
  </Card.Header>
  <Card.Section>
    <Card.Section.Text>
      Some quick example text to build on the card title and make up the bulk of
      the card's content.
    </Card.Section.Text>
  </Card.Section>
</Card>
```

### Card.Header

##### Basic Usage

```jsx live
<Card>
  <Card.Header>
    <Card.Header.Title>Card Title</Card.Header.Title>
  </Card.Header>
</Card>
```

##### Small Variant
```jsx live
<Card>
  <Card.Header>
    <Card.Header.Title small>
      Card Title
    </Card.Header.Title>
  </Card.Header>
</Card>
```

##### Custom Title Sizes
```jsx live
<Card>
  <Card.Header>
    <Card.Header.Title sizes={{ default: 'h2', small: 'h3' }}>
      Card Title
    </Card.Header.Title>
  </Card.Header>
</Card>
```

### Card.Section

##### Basic Usage

```jsx live
<Card>
  <Card.Section>
    <Card.Section.Title>
      Section title
    </Card.Section.Title>
    <Card.Section.Content>
      <Card.Section.Text>
        This is a card section. It can contain anything but usually text, a list, or list of links. Multiple sections have a card divider between them.
      </Card.Section.Text>
    </Card.Section.Content>
    <Card.Section.Actions>
      <Card.Section.Action href="https://edx.org">
        Action 1
      </Card.Section.Action>
      <Card.Section.Action href="https://edx.org">
        Action 2
      </Card.Section.Action>
    </Card.Section.Actions>
  </Card.Section>
</Card>
```

### Card.Divider

##### Basic Usage

```jsx live
<Card>
  <Card.Section>
    Some text goes here.
  </Card.Section>
  <Card.Divider />
  <Card.Section>
    Some text goes here.
  </Card.Section>
</Card>
```
