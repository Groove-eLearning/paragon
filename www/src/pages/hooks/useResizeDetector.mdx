# useResizeDetector

<SingleComponentStatus
  frontmatter={
    get(props, 'pageContext.frontmatter') ? props.pageContext.frontmatter : null
  }
/>

<p>
  This hook gives quick access to the dimensions (width/height) of any DOM
  element in a performant way. Change the screen size to see the effect.
</p>

<p>
  See
  {' '}
  <a href="https://www.npmjs.com/package/react-resize-detector" target="_blank" rel="noopener noreferrer">
    https://www.npmjs.com/package/react-resize-detector
  </a>
  {' '}
  for full documentation.
</p>

#### Basic Usage

```jsx live
() => {
  const { height, width, ref } = useResizeDetector();

  return (
    <div ref={ref} className="border p-3">
      <div>Height: {height}px</div>
      <div>Width: {width}px</div>
    </div>
  );
}
```

#### Optimizing for performance

<p>This hook may be configured to be throttled or debounced.</p>

```jsx live
() => {
  const { height, width, ref } = useResizeDetector({
    refreshMode: 'throttle',
    refreshRate: 200,
  });
  return (
    <div ref={ref} className="border p-3">
      <div>Height: {height}px</div>
      <div>Width: {width}px</div>
    </div>
  );
}
```
